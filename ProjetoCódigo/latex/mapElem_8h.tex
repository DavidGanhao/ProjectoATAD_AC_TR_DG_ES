\doxysubsection{Detailed Description}
Defines the types Map\+Key and Map\+Value. 

The Map\+Key is an alias to the type of keys held by an instance of the A\+DT Map (Pt\+Map).

The Map\+Value is an alias to the type of values held by an instance of the A\+DT Map (Pt\+Map).

These aliases just be changed according to the use-\/case.
